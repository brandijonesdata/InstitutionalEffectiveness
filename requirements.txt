# Georgia Institutional Effectiveness Dashboard

Interactive decision-support dashboard for context-adjusted
completion, Pell equity, resource adequacy, institutional
profiles, and empirical peer comparisons.

## Repository structure

```text
InstitutionalEffectiveness/
├── streamlit_app.py
├── requirements.txt
├── README.md
└── data/
    ├── georgia_dashboard_data.csv
    └── dashboard_build_audit.csv
```

## Run locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Interpretation

Georgia institutions were excluded from national model fitting.
Results are institution-level predictive associations intended
for diagnostic planning rather than punitive ranking.
